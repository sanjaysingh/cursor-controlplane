Release v0.7.0 - Telegram command improvements

- Added /model-default command to show current default model
- Added /session-current command to show session details
- Added /session-close-all command to close all sessions
- Renamed commands to consistent noun-action pattern:
  - /sessions -> /session-list
  - /models -> /model-list
  - /workspaces -> /workspace-list
  - /repos -> /repo-list
  - /session_close -> /session-close
- Removed /status command (duplicate of /session-list)
- Removed /session_new command (auto-create on first message)
- Auto-create session at workspace root when message sent without active session
